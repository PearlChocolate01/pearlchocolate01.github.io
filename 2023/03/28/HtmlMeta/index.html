HTML中的meta标签

发表于2023-03-28|更新于2023-11-26|计算机
|字数总计:379|阅读量:

定义

meta标签用来提供有关html文档的元数据，用户看不到，蛋浏览器可以识别，对SEO（搜索引擎优化）有重要作用。

常用meta标签列举

1. author

表示网页作者的名字、组织、机构。

2. description

一段简短的，对页面的描述。
一般显示在搜索引擎中对网页的描述。但是搜索引擎也会抓取网页中最匹配的一段文字显示。
怎么写：一到两个关键词，150字左右，添加关键信息，内容匹配度高，适当增加一些“learn more”等引导词。

3. keywords

与页面相关的关键词，部分搜索引擎会根据这些关键词对文章进行分类。
目前对于SEO不一定有用了（关键词的堆砌有讨好搜索引擎的嫌疑）。

4. viewport

viewport(视口)是指设备屏幕上能用来显示网页的那一块区域。
为视口的初始大小提供指示，仅用于移动设备。


width设置viewport的宽度为设备宽度（css像素），device-width是指这个设备最理想的viewport宽度；
initial-scale为设备宽度与viewport大小之间的缩放比例。
